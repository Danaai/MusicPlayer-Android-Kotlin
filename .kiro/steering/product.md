# Product Overview

## Music App

A modern Android music streaming application built with Jetpack Compose. The app is designed to provide a comprehensive music listening experience with features for browsing, playing, and managing music content.

## Key Features
- Music streaming and playback using ExoPlayer
- Modern Material Design 3 UI
- Local data persistence with Room database
- Network connectivity for music content via Retrofit
- Dependency injection architecture with Hilt
- Image loading and caching with Coil

## Target Platform
- Android devices (API 24+)
- Supports dynamic theming on Android 12+
- Edge-to-edge display support
