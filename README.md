# 🎵 Music Player App (Android - Kotlin + Room)

## 📱 Overview

MusicApp is a mobile application that allows users to browse, play, and manage music locally. The app integrates both user and admin functionalities in a single system, enabling efficient data management and a seamless music listening experience.

---

## 🚀 Features

### 👤 User Features

* Browse and play music
* Add/remove favorite songs
* Search songs by name
* View song details

### 🛠 Admin Features

* Add, edit, delete songs
* Manage music library
* Store and update data using Room Database

---

## 🛠 Tech Stack

* **Language:** Kotlin
* **UI:** Jetpack Compose 
* **Architecture:** MVVM
* **Database:** Room
* **Components:** ViewModel, StateFlow / LiveData

---

## 🧠 Architecture

The app follows MVVM architecture:

* UI layer (Compose)
* ViewModel handles state and business logic
* Repository manages data from Room database

---

## 🎯 Key 

* Designed and implemented music player UI
* Integrated Room database for local data persistence
* Built CRUD system for music management (Admin + User)
* Applied MVVM architecture for clean and scalable code
